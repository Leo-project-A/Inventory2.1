<!-- GETTING STARTED -->
## Getting Started

1. Install dependencies
   ```sh
   pip install -r requirements.txt
   ```
2. Run the program
   ```sh
   python GUI.py
   ```

<!-- USAGE EXAMPLES -->
## Usage

1. Create an inventory
2. Add items
3. review and update stocks, to know which items are running low
4. edit the inventory
